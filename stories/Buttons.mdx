## マージン

| 余白  | 説明                                            |
| ----- | ----------------------------------------------- |
| `p-s` | ボタンには縦・横 それぞれ16pxのパディングを設定 |

## ターゲット領域

ターゲットは、指のタッチやマウスのようなポインティングデバイスによるポイントアクションを受け入れる領域を指します。

## ボタン幅

ボタンの幅は、スマートフォンの場合は、上記のコンテンツと等倍とし、
タブレット以降はフォームに付随するボタンの場合はフォーム幅と等倍とします。

| 指定方法                      | 説明                                               |
| ----------------------------- | -------------------------------------------------- |
| `className`に幅のクラスを指定 | `<Button className="col-span-full" />`のように設定 |
