# リンクテキスト

文中のリンクを表現する場合、リンクカラーを用いてテキストには下線を引きます。
リンクの色は、カラーガイドを踏襲してください。

[デザインシステム｜デジタル庁](https://www.digital.go.jp/policies/servicedesign/designsystem/)では、リンクテキストはスタイルに分類されていますが、
これは多分コンポーネントでしょう。

詳細は、"コンポーネント/リンクテキスト"を参照してください。

## ガイドライン

### リンクの目的を明確に

利用者がリンクの先を訪れたいか決めれあること、スクリーンリーダーで理解しやすいことをサポートします。

リンク先のURLにある情報を説明するテキストをテキストリンク自体や、テキストリンクの前に含んでください。

### 外部リンク・ファイル形式

新規ウィンドウを開くリンクには、リンク末尾にアイコンをつけて、その旨を喚起します。

なお、外部サイトへの遷移となる外部リンクは、減速として新規ウィンドウで開きます。

PDF資料等のドキュメントリンクを示す場合は、リンクテキストにファイル形式及び容量を明記します。

## 仕様

### モバイルデバイスへの対応

主要なボタンやナビゲーションなどは、クリックやタップの誤操作を防ぐため、`44x44 CSS px`以上の大きさとします。

しかし、インラインのリンクテキストの場合は、クリックやタップ可能なターゲット領域は、
隣接するターゲット領域と重複がない状態で少なくとも`24x24 CSS px`以上とします。
