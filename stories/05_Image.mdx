# 画像

[デザインシステム｜デジタル庁](https://www.digital.go.jp/policies/servicedesign/designsystem/) を参考に、
アプリで画像のアスペクト比を選びます。

## アプリで画像のアスペクト比を選びます。

動画やイメージ画像のアスペクト比は、`16:9`、
写真や一部のスクリーンなどでは `3:2`の一般的なアスペクト比を使用、
人物紹介等の画像比は1:1とします。

バナーは国際標準バナーサイズであるハーフバナー(`234x60IMU`)の使用を推奨します。

定期刊行物や文章など物理的な寸法が決まっている場合(A4サイズやB5サイズなど)は
`16:9`や`1:1`に無理に当て込まず、プロダクトそのものの比率を尊重します。
